## 0.1.0 - initial release

* Introduces the following widgets:
  * `GraphViewport`
  * `GraphView`
  * `NodeWidget`
  * `EdgeWidget`
* Introduces styling with:
  * `GraphStyle`
  * `NodeStyle`
  * `EdgeStyle`
* Built-in viewport panning, scaling and scrolling
* Built-in node and node group dragging
